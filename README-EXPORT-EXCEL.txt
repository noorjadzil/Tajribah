MAQSHOF - Versi Pilihan Export Excel

Cara pakai:
1. Upload semua file ini ke repository GitHub Pages kamu.
2. Buka aplikasi seperti biasa.
3. Masuk tab Laporan.
4. Coba tombol:
   - Excel 1 CSV: paling ringan, cocok untuk data mentah.
   - Excel 2 XLS Ringan: tabel rapi ringan, ekstensi .xls.
   - Excel 3 XML Rapi: lebih rapi, format Excel XML.
   - Excel 4 XLSX Asli: file .xlsx asli tanpa CDN/library luar.

Catatan:
- Kalau HP/browser memblokir download, coba buka lewat Chrome.
- Untuk hasil paling stabil di HP biasanya Excel 1 atau Excel 3.
- Untuk hasil paling modern coba Excel 4.
