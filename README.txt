CARA PAKAI DI GITHUB PAGES

1. Buat repository baru di GitHub.
2. Upload semua file ini:
   - index.html
   - style.css
   - app.js
   - particles.js
   - folder assets
3. Buka Settings > Pages.
4. Pilih Branch: main, folder: /root.
5. Save.
6. Tunggu link GitHub Pages muncul.
